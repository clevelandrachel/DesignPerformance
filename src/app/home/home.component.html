Welcome

Design for Performance was created from the evolving need of software technology to continually strive to discover and address the needs of human computer interaction. The designs discussed can be applied to any web technology, regardless of the underlying framework.

This site was inspired by and built with Project Clarity, an open source design system that brings together UX guidelines, HTML/CSS framework, and Angular components. More to Come on this Site Spring 2019.

How to Use This Guide

There are three approaches for utilizing Design for Interaction: just the Interaction guidance, Interaction guidance and the HTML/CSS/JS framework, or put everything together with the Clarity Design System.

About Design for Performance

Understanding user experience in the context of interaction design can aid in the ability to detect and recommend ways to improve performance in application design. In turn, improving the usability of application design can increase customer satisfaction in products.

The guidance outlined in this document relates to the performance of user interaction focused on instrumenting systems and applications to collect metrics and logs and then validating the architecture through user insights.

More background and resources are provided on the site.
